# 📚 Документация Meet-Commit

## 🎯 О проекте

**Meet-Commit** — это интеллектуальная система автоматической обработки встреч с помощью искусственного интеллекта. Система превращает любые файлы встреч в структурированные отчеты с автоматическим извлечением ключевой информации, управлением обязательствами и сохранением в Notion.

## 🚀 Основные функции

### 📄 Обработка встреч
Автоматическая суммаризация файлов встреч (PDF, DOCX, TXT, VTT) с извлечением участников, дат и ключевой информации.

### 🏷️ Система тегирования
Интеллектуальное присвоение тегов встречам на основе содержимого с возможностью интерактивного редактирования.

### 📝 Управление коммитами
Автоматическое извлечение и отслеживание обязательств из встреч с системой проверки качества.

### 👥 Управление людьми
Автоматическое обнаружение и верификация участников встреч с пополнением словаря людей.

### 📊 Система повесток
Автоматическая генерация структурированных повесток на основе активных задач и обязательств.

### 🔍 Быстрые запросы
Мгновенный поиск и фильтрация коммитов по различным критериям через Telegram бота.

## 🎯 Быстрый старт

### 👤 Для пользователей

1. **📎 Загрузите файл встречи** в Telegram бота
2. **🎛️ Выберите стиль суммаризации** (Brief/Detailed/Structured)
3. **🏷️ Отредактируйте теги** через интерактивное ревью
4. **📊 Используйте повестки** для планирования и отчетности
5. **🔍 Ищите коммиты** через команды быстрого доступа

**Основные команды:**
```bash
/start              # Начало работы с ботом
/help               # Справка по командам
/process            # Инструкция по обработке файлов

# Создание коммитов
/commit             # Интерактивное создание
/llm <текст>        # Создание через естественный язык

# Поиск коммитов  
/mine               # Мои задачи
/due                # Дедлайны на неделю
/by_tag <тег>       # Поиск по тегу

# Повестки
/agenda             # Интерактивное создание
/agenda_person <имя> # Персональная повестка

# Управление людьми
/people_miner2      # Верификация новых людей
```

### 🔧 Для администраторов

1. **🔐 Настройте админский доступ** - следуйте [инструкции](./setup_admin_access.md)
2. **📊 Мониторьте систему** - используйте `/metrics` и `/tags_stats`
3. **👥 Управляйте людьми** - регулярно проверяйте `/people_miner2`
4. **🏷️ Контролируйте теги** - валидируйте правила через `/tags_validate`

**Административные команды:**
```bash
/admin_help         # Все административные команды
/metrics            # Метрики производительности
/tags_stats         # Статистика тегирования
/people_stats_v2    # Статистика системы людей
/tags_validate      # Валидация правил тегирования
/retag_all          # Массовое перетегирование встреч/коммитов
/retag_help         # Справка по перетегированию
```

### 👨‍💻 Для разработчиков

1. **📖 Изучите архитектуру** - начните с [технической документации](./tagger_v1_scored.md)
2. **🧪 Запустите тесты** - `pytest tests/ -v`
3. **🔍 Используйте CLI утилиты** для отладки
4. **📊 Настройте мониторинг** производительности

**Полезные команды:**

## 📊 Архитектура системы

### Основные компоненты

```
📱 Telegram Bot (aiogram)
        ↓
🧠 AI Processing (OpenAI GPT-4o-mini)
        ↓
🏷️ Dual Tagging System (Token + Regex)
        ↓
👥 People Management (Detection + Verification)
        ↓
📝 Commits Extraction & Validation
        ↓
💾 Notion Integration (Structured Storage)
```

### Технологический стек

- **Backend:** FastAPI + aiogram (Telegram Bot API)
- **AI:** OpenAI GPT-4o-mini для суммаризации и извлечения
- **Storage:** Notion как основная база данных
- **Processing:** pdfminer, python-docx, webvtt для извлечения текста
- **Quality:** pytest (466 тестов, 75% покрытие), ruff, mypy, bandit
